## [1.0.0] - 2024-12-13

### Добавлено
- Первая рабочая версия симулятора Solar Orbital Simulator
- Физическая модель на основе закона всемирного тяготения Ньютона
- Реализация метода Velocity Verlet для численного интегрирования
- Визуализация движения Солнца и Земли в реальном времени
- Настраиваемое масштабирование для отображения космических расстояний
- Интерактивное управление камерой (зум и панорамирование)
- Графический интерфейс на основе Qt6
- Поддержка множественных небесных тел
- Система сборки с использованием CMake
- Управление зависимостями через vcpkg
- Готовые исполняемые файлы с включенными зависимостями

### Технические детали
- **Физические расчеты**: Закон всемирного тяготения с двойной точностью
- **Интегрирование**: Velocity Verlet метод для стабильности орбит
- **Графика**: Qt Graphics View Framework для рендеринга
- **Масштабирование**: 100 пикселей = 1 астрономическая единица
- **Временные шаги**: 1 день симуляции за ~16мс реального времени
- **Частота кадров**: 60 FPS

### Параметры системы по умолчанию
- **Солнце**: Масса 1.989×10³⁰ кг, радиус 696,340 км, цвет желтый
- **Земля**: Масса 5.972×10²⁴ кг, радиус 6,371 км, цвет синий
- **Начальное расстояние**: 1 АЕ (149.6 млн км)
- **Орбитальная скорость**: ~30 км/с

### Зависимости
- C++17
- Qt 6.x
- Eigen3 (математические вычисления)
- CMake 3.16+
- Visual Studio 2019/2022

- Только 2D визуализация (планируется 3D)

- Отсутствует сохранение состояния симуляции
- Нет экспорта данных о траекториях

## Формат версий

Мы используем [Semantic Versioning](https://semver.org/):

- **MAJOR** (X.y.z): Несовместимые изменения API
- **MINOR** (x.Y.z): Новая функциональность, совместимая с предыдущими версиями
- **PATCH** (x.y.Z): Исправления ошибок, совместимые с предыдущими версиями

## Типы изменений

- **Добавлено** - для новых функций
- **Изменено** - для изменений в существующем функционале
- **Устарело** - для функций, которые будут удалены в будущих версиях
- **Удалено** - для удаленных функций
- **Исправлено** - для исправления ошибок
- **Безопасность** - для исправлений уязвимостей
